[SQL] 데이터 조회 SELECT

# SELECT - 칼럼명에 *를 넣으면 모든 칼럼을 조회할 수 있다. ``` SELECT `칼럼명` FROM `테이블명`; ``` # SELECT, WHERE (조건추가) - 뒤에 WHERE를 붙이면 원하는 조건의 데이터만 조회할 수 있다. ``` SELECT `칼럼명` FROM `테이블명` WHERE `칼럼` = `데이터`; ``` # 예시 ``` SELECT * FROM article WHERE id = 1; ``` - 이렇게 쓴다면 id가 1인 row의 모든 칼럼들이 조회된다.
